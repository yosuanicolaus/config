### [260213] For config files in home (`~`)

Run these if they already exist (removes them)

```
rm ~/.ideavimrc
rm ~/.zshrc
```

Run these to make the symlink

```
ln -s ~/.config/yosuanicolaus/.ideavimrc ~/.ideavimrc
ln -s ~/.config/yosuanicolaus/.zshrc ~/.zshrc
```
